# 로또 번호생성 프로그램 Java로 구현해보기

## 1. 기능
- 번호 생성하기
  - 번호합 구간 입력
  - 연속 미출현 번호 제외
  - 역대 당첨번호 제외
  - 번호 생성
- 통계 조회
  - 이번 주 당첨번호
  - 역대 1등 당첨번호의 번호합
  - 역대 1등 당첨번호중 연속 미출현 번호 및 정보
- 경우의 수 조회
  - 1등 당첨 경우의 수
  - 번호합 별 경우의 수

## 2.  동작 순서
1. 금주 회차 입력
   1. 회차 입력시 숫자가 아닌 문자가 있는지 체크 (정규식)
   2. 회차 입력시 Integer 범위 안으로 체크 위해 10억 미만의 수인지 체크
   3. 오 입력시 다시 입력하도록 유도
2. 역대 당첨번호 입력
3. 동작 선택
   1. 번호 생성
   2. 통계 조회
   3. 경우의 수 조회
4. 종료
